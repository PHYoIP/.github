## PHYoIP

Physical interfaces like UART over TCP/IP.

#### Client Types
|  Type   | Description                                           |
|:-------:|:------------------------------------------------------|
| __GC__  | Gateway (needed if the PHY is not on the server host) |
| __RWC__ | Read/write client                                     |
| __SC__  | Sniffer client (receives ingress and egress packets)  |
